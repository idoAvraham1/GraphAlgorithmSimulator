Build your graph

Use the toolbar to switch between editing modes:

  • + Node — click empty space on the canvas to place a node.
  • Connect — click two nodes to link them with a weighted edge.
  • Edit edge — click an edge to change its weight or remove just that edge.
  • Delete node — click a node to remove it (edges attached to it go too).
  • Click any tool again (or press Esc) to exit editing and pan or drag nodes freely.

Try the Preset dropdown for example graphs, or build your own.

Choose an algorithm

Pick BFS, DFS, Dijkstra, or Floyd–Warshall from the top bar. Set a start node for single-source algorithms, and toggle Directed if edges should have direction.

Run and learn

Press Run, then use play/pause or step buttons to watch nodes and edges highlight on the canvas. Use the Steps tab to follow along; Distances (Dijkstra) and All pairs (Floyd–Warshall) show extra detail.
